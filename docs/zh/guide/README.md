# toolbox 是什么?

`toolbox` 一个用于集成常用函数或不常用的函数插件库，使用简单，便捷和兼容性强。

### 它能为我做什么

toolbox 一个用于集成常用函数或不常用的函数插件库，集成了常见函数或不常见函数，方便用于平时开发。
